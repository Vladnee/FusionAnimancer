# Getting Started

Follow these steps to set up and run the sample:

1. Open the project in **Safe Mode**.  
2. Open the **Package Manager**.  
3. Import **Photon Fusion 2**.  
4. Import **Animancer** (Lite or Pro).  
5. In **PhotonAppSettings**, set your **Fusion App Id**.  
6. In **NetworkProjectConfig**, set **PeerMode** to **Multiply**.  
7. Run the **SampleScene**.
